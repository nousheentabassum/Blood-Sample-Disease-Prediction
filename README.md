# 🩺 Blood Sample Disease Prediction
     
## Overview
Machine learning project to predict diseases like Anemia, Diabetes, and Thalassemia using blood test data.

## Features
- Data preprocessing and scaling
- Multiple ML models (Logistic Regression, Random Forest, Gradient Boosting, SVM)
- Model comparison using accuracy, precision, recall, F1 score
- Confusion matrix visualization
- Interactive prediction using user inputs

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## How to Run
1. Open in Google Colab or Jupyter Notebook
2. Upload dataset CSV
3. Run all cells

## Results
- Best model: SVM
- Accuracy: ~53%
- F1 Score: ~0.55
